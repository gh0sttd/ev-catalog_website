# ⚡️ EV Catalog — Система керування автосалоном

### 🚀 Про проєкт
Це повнофункціональний веб-додаток для продажу електрокарів, побудований на архітектурі **MVT (Model-View-Template)**. Система включає повний цикл взаємодії: від перегляду характеристик до запису на тест-драйв.

### 🛠 Технічний стек
- **Framework:** Django 5.0
- **Database:** SQLite з оптимізованими індексами (`db_index=True`)
- **Frontend:** Tailwind CSS (Modern UI/UX)
- **Deployment:** Підтримка Gunicorn для Production-середовища

### 🏗 Архітектура бази даних
- **Car:** Зберігає технічні дані та медіа-файли.
- **Favorite:** Реалізує зв'язок Many-to-Many через проміжну модель для персоналізації.
- **TestDriveRequest:** Система логування клієнтських запитів з прив'язкою до `User`.

### 🛡 Безпека та Оптимізація
- Захист від CSRF-атак у всіх формах.
- Валідація вводу даних на рівні моделей Django.
- Використання `select_related` для зменшення кількості SQL-запитів до бази.

### ⚙️ Встановлення та запуск
1. Встановіть залежності: `pip install -r requirements.txt`
2. Виконайте міграції: `python manage.py migrate`
3. Запустіть сервер: `python manage.py runserver`
